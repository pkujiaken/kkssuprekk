预测市场平台如Kalshi，此前主要吸引大众押注体育和流行文化事件，现在正寻求将华尔街机构交易员作为其下一个主要客户群体。他们希望机构投资者能利用这些平台进行大额交易，例如对冲策略。这一趋势预示着预测市场可能从边缘走向主流金融领域，为机构投资者提供一种新型的风险管理和市场洞察工具。投资者应关注这种金融创新如何改变传统对冲和风险评估方式，以及其潜在的市场容量和监管挑战。

美国生物技术或迎复兴，中美竞争加剧改变全球格局
美国蒙大拿州通过的SB535法案旨在吸引生物技术投资，这可能为美国生物技术产业带来新的复兴。与此同时，中国在新药审批数量和临床试验全球份额方面迅速增长，在2024年已超越阿斯利康成为全球最大的临床试验发起方，并接近甚至超越美国。这种结构性变化显示了全球生物技术创新和产业重心正在发生转移，中美之间的科技竞争已扩展到生命科学领域。投资者应关注这一趋势对全球医药研发、创新生态和相关投资机会的长期影响。

金融监管机构SEC面临削弱，对市场长期健康构成潜在风险
美国金融监管机构证券交易委员会（SEC）正经历资金减少和监管标准模糊化的压力，其自身的委员也被指责正在削弱该机构的影响力。这种对华盛顿最强硬金融监管机构的削弱，可能导致市场缺乏足够的监督和制衡，从而增加金融系统的风险。尽管短期内可能被市场对AI等热点的追逐所掩盖，但长远来看，监管疲软可能侵蚀市场诚信、助长不当行为，最终损害投资者利益和资本市场的长期健康发展。
