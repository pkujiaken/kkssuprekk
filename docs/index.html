市场竞争与计算复杂性理论的深层关联
经济学家Tyler Cowen提出，市场是否具有竞争力可能取决于一个深层的计算复杂性理论问题——P与NP是否相等。如果P=NP，企业能够高效地检测和维持串谋，从而削弱市场竞争性。这一理论挑战了传统经济学对竞争市场的理解，揭示了市场结构背后潜在的计算限制。
