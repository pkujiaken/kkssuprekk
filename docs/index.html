在市场对私有信贷（private credit）市场担忧加剧之际，分析师警告投资者应警惕风险更高的私有股权基金。这些基金持有与信贷市场同样面临困境的非上市公司资产，其投资者正处于潜在风险的中心。当前环境下，私有市场投资的透明度较低和流动性不足，可能导致投资者面临更大的不确定性，尤其是在经济下行周期中。
