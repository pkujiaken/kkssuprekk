美联储就稳定币客户识别计划征求意见，预示加密资产监管提速
美国联邦储备委员会提议要求特定支付稳定币发行商建立有效的客户识别计划（CIP），并就此向公众征求意见。这一举措标志着监管机构正积极将加密资产纳入传统金融监管框架。此提案旨在提高稳定币交易的透明度，防范洗钱和恐怖主义融资风险。其重要性在于，它为未来对加密货币的全面监管奠定了基础，将对稳定币的发行、交易和市场采用产生长期影响，是加密市场走向成熟和规范化的关键一步。
