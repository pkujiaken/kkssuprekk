美国人口普查局禁止“噪音注入”统计产品，提升数据透明度
美国人口普查局决定禁止在统计产品中“注入噪音”，以提高数据透明度和准确性。此举意在解决在保护隐私的同时，如何提供高质量公共数据的问题。对于依赖官方统计数据进行宏观经济分析和投资决策的专业人士而言，这一政策变化将确保数据来源的可靠性，减少潜在的误差和误导，对理解经济真实状况具有重要意义。
