Z世代对国家养老金制度信心不足，正积极规划私人养老
一项调查显示，许多Z世代（Generation Z）的年轻人不相信国家养老金在他们年老时还会存在，因此他们正积极规划自己的私人养老金。这一趋势反映了年轻人对未来社会保障体系的深层担忧，以及个人财富管理观念的转变。对金融服务业和长期投资市场而言，这意味着对个人养老产品、储蓄和投资工具的需求将持续增长，特别是针对年轻群体的创新型养老解决方案。这一代人的财务规划模式将对未来的资产管理行业产生结构性影响。
