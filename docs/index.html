AI热潮下青年失业率保持不变，但市场竞争与计算复杂性挂钩
观点：经济学家Tyler Cowen援引Conor Sen的观察指出，自AI热潮开始以来，20-24岁年龄段的失业率几乎没有变化。同时，Cowen还探讨了一项理论，即市场是否具有竞争力取决于计算的复杂性（P!=NP）。如果P=NP，则企业能有效识别串通行为，使串通成为可持续均衡。这表明AI对劳动力市场的影响可能比预期更复杂，而市场效率与计算能力的深层关系值得高净值投资者关注。
