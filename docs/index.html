警惕私人信贷和私募股权的风险，比预期更具波动性
报告警告称，如果投资者担心私人信贷市场的风险，那么应该远离风险更高的私募股权投资。私募股权基金持有与信贷市场同样表现不佳的公司，其投资者面临更高风险。这提醒投资者在追求高回报的同时，需警惕这些非公开市场中固有的流动性和估值风险。为什么重要：私人市场资产的风险敞口日益增加，对高净值投资者而言，了解这些资产的真实风险和潜在的链式反应至关重要。
